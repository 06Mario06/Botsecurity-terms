Privacy Policy

Nexus Security – Discord Moderation & Security Bot

1. Data Collected

Nexus Security may collect technical data required for its operation, including user IDs, server IDs, message IDs, timestamps, and moderation logs. No private message content is intentionally collected.

2. Purpose of Data Processing

Data is processed exclusively for moderation, server security, abuse prevention, and enforcement of server rules.

3. Data Retention

Data is retained only for the time strictly necessary for the bot’s functionality. Data may be deleted upon request by the server administrator or the affected user.

4. Data Sharing

Data is not sold, shared, or disclosed to third parties and is never used for advertising or profiling purposes.

5. Data Security

Reasonable technical and organizational measures are implemented to protect data against unauthorized access, loss, or misuse.

6. User Rights

Users may request access to or deletion of their data, subject to moderation and security requirements.

7. Contacts

For privacy-related inquiries or data removal requests, contact the bot owner via email or Discord.
